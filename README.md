# Global Trade Risk Map

한국 수출입 기업이 마주하는 **글로벌 무역 리스크를 공개 데이터로 연결·시각화**하는 웹 인텔리전스 데스크입니다.

**▶ 라이브 데모: https://krkr5628.github.io/global_trade_risk_map.github.io/**

이 저장소는 위 주소로 서비스되는 **정적 프론트엔드 빌드(React/Vite)** 를 담고 있습니다.

---

## 목적

뉴스, 공시, 공공 통계, 국제 기업정보 등 흩어져 있는 공개 데이터를 **기업 → 품목(HS) → 국가 → 리스크 이벤트**의 연결 그래프로 엮어, 한국 무역기업의 교역 리스크를 한 화면에서 근거와 함께 파악할 수 있게 하는 것이 목표입니다.

- 특정 기업이 어떤 국가·품목에 노출되어 있는지
- 그 국가/경로에서 어떤 리스크 이벤트·규제·수출통제가 발생 중인지
- 단기·중기·장기로 어느 정도 영향이 예상되는지

를 **원천 데이터 근거(evidence)** 와 함께 제시합니다.

## 동작 방식

1. **수집·정규화** — 아래 공개 데이터를 수집해 국가 코드(ISO2)·HS 코드·기업 식별자 기준으로 정규화합니다.
2. **엔티티 해소** — 기업명/DART 계약 텍스트에서 취급 품목(HS)을 후보로 추정하고, 국내 기업을 GLEIF LEI·글로벌 소유그래프와 연결합니다.
3. **노출 결합** — 기업의 HS·국가 노출을 국가별 무역흐름·리스크 이벤트·공개 지표와 결합해 노출 엣지(edge)를 만듭니다.
4. **영향 산출** — 결합된 노출에 리스크 이벤트·수출통제·수입규제·거시(환율 등)를 가중해 단기/중기/장기 영향 벡터와 점수를 계산합니다.
5. **근거 제시** — 모든 점수는 클릭 시 원천 이벤트·공시·통계 row까지 추적할 수 있도록 근거 테이블·그래프로 표시합니다.

> 표시되는 공급망·경로·품목 노출은 확정 사실이 아니라 **공개 데이터 기반 후보(candidate) 신호**이며, 각 화면에 그 한계를 명시합니다.

## 주요 기능 (화면별)

### 🗺 Trade Risk Map
국가별 리스크를 지도 위에서 조망합니다. 글로벌 뉴스 이벤트, 공개 국가 리스크 지표, 선택한 HS 품목의 국가별 교역 흐름을 Leaflet 지도와 국가 랭킹으로 함께 표시해, "어느 국가가 지금 위험 신호가 높은가"를 한눈에 파악합니다.

### 📰 News & Signals
무역 관련 리스크 이벤트 스트림입니다. 뉴스 이벤트를 국가·리스크 팩터별로 요약하고, 공개 국가지표와 함께 **기업별 단기/중기/장기 영향 벡터**로 환산해 보여줍니다. 개별 이벤트에서 어떤 국가·품목·기업으로 신호가 전파되는지 추적할 수 있습니다.

### 🏢 Korea Company Exposure
국내 기업 관점의 노출 뷰입니다. DART 계약 노출과 최근 공시 이벤트, 기업별 계약 요약을 보여주고, 선택 기업의 **품목(HS) 기반 후보 국가 노출**을 함께 제시합니다. 계약 금액 필드가 접수번호 등으로 오염된 건은 품질 플래그로 분리해 신뢰도를 구분합니다.

### 🔗 Risk Ontology Graph
"기업 → HS 후보 → 국가 → 리스크 이벤트 → 리스크 팩터 → 영향 벡터"로 이어지는 연결 경로를 **온톨로지 그래프**와 근거 테이블로 시각화합니다. 하나의 리스크가 왜/어떻게 특정 기업까지 연결되는지 경로 단위로 설명합니다.

### 📊 Impact Reporting
선택 기업 기준의 종합 리포트 화면입니다. 단기/중기/장기 영향 점수, top 영향 경로, DART 계약·공시, 뉴스 리스크 이벤트, 공공데이터/HS 교역 흐름 근거, 그리고 **보고서용 문구 초안**을 한 화면에 모아 제시합니다.

### 🌐 Impact Propagation
선택 기업과 HS·국가·리스크 축을 공유하는 **연관 기업(affected peers)** 과 공급 클러스터 후보를 표시합니다. 확정 공급망이 아니라 공유 노출(shared exposure) 기반 후보로 관리합니다.

### 🚢 Route Corridor Risk
교역 경로 리스크 분석입니다. 부산 출발 해상 16개 경로와 인천(ICN) 출발 항공 6개 경로, 주요 해상 chokepoint(예: IMF PortWatch 계열) 12개를 지도에 표시하고, 뉴스 리스크 이벤트가 경로의 10–50km 버퍼와 교차하는지, 인접국 위험 평균, 기업별 경로 노출을 분석합니다. 실제 선박·항공편 추적이 아니라 **경로 후보(route candidate) 기반 리스크 노출**입니다.

### 🛰 Public Data Radar
데이터/근거 상태 모니터입니다. 캐시·근거 데이터의 최신성, 공개 데이터 소스별 준비 현황(readiness)을 표시하고, 여기서 선택한 기업이 다른 화면과 공유됩니다.

> 공통 UX: 상단 command bar의 기업 입력과 화면별 기업 선택이 Impact Reporting·Risk Ontology Graph 등 주요 화면 전반에 공유되어, 하나의 기업을 여러 관점으로 연속 분석할 수 있습니다.

## 사용한 데이터

모두 **공개(공공/오픈) 데이터**를 사용합니다.

| 구분 | 소스 | 활용 |
| --- | --- | --- |
| 무역·리스크 이벤트 | 글로벌 뉴스 이벤트 데이터(GDELT) | 국가·품목 리스크 이벤트, 영향 벡터 |
| 국내 기업 공시 | 금융감독원 DART | 계약 노출·공시 이벤트 |
| 무역보험·수출입 | K-SURE 국가/업종 위험지수 | 국가·업종 위험 blend |
| 해외시장·진출기업 | KOTRA 해외시장뉴스, 해외 한국기업 디렉토리, 국별 수입규제 | 공식 뉴스, 기업 진출국, 수입규제 근거 |
| 수출통제 | 전략물자관리원 HSK 연계표 | 전략물자 수출통제 근거 |
| 통계·거시 | 통계청 KOSIS 수출입, 한국은행 ECOS(환율·금리) | 교역 가중치, 환·결제 리스크 맥락 |
| 국제 지표 | World Bank, US Census HS 무역, IMF PortWatch 해상 chokepoint | 공개 국가 리스크·교역 흐름·경로 |
| 글로벌 기업정보 | GLEIF LEI, 미국 SEC EDGAR, 일본 국세청 법인번호·EDINET | 글로벌 엔티티/소유그래프 해소 |

각 데이터는 정규화 후 기업–품목–국가 축으로 결합되어 리스크 점수와 근거 리포트에 반영됩니다.

## 구성

- **Frontend (이 저장소)**: React/Vite 정적 빌드. GitHub Pages로 배포됩니다.
- **Backend / 데이터 파이프라인**: 공개 데이터 수집·정규화·리스크 점수 산출 로직으로, 이 공개 저장소에는 포함되지 않습니다.

## 데이터 출처 및 라이선스

각 데이터의 권리는 원 제공기관에 있으며, 본 프로젝트는 공개된 이용 조건에 따라 분석·시각화 목적으로 활용합니다. 표시되는 리스크 점수·영향 벡터는 분석적 추정치이며 투자·거래 판단의 근거로 보증되지 않습니다.
